# Workbench AI

AI-powered product definition platform.

## Features

- PM Discovery with AI agents
- Canonical Architecture generation
- INVEST-compliant backlog creation
- Read-only repository introspection (REPO-100, REPO-101)

## Development
```bash
# Run server
uvicorn app.main:app --reload

# Run tests
pytest tests/ -v
```
